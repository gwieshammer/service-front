Should this Todo be a simple one?
If this box is unchecked this Todo will be complex and offer more options. This means, you can add a description to a todo-item. In the future you will be able to set a duedate, so you'll get a reminder either via email or a text message on your cellphone. If this Todo is simple, you can only add items to your todo. And if the entire todo is done the process will get completed with out any further action. A complex todo gets a final task (Finish + the name of the todo) where you can definitely say that everything is done. If not the Todo Process is going back so one can continue the work.
